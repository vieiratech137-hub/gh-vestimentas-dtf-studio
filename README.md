# GH Vestimentas DTF Studio

Site estático público para manipulação de imagens voltado para estamparia DTF.

## Recursos

- remoção de fundo em lote no navegador
- editor de cores com paleta detectada automaticamente
- exportação para PNG, PDF e DTF-ready em 300 DPI
- upload por clique ou drag and drop
- processamento local, sem backend

## Publicação

Este repositório foi preparado para GitHub Pages com os arquivos estáticos na raiz.
